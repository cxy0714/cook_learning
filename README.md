# 🍳 做饭练习

我的学做饭仓库：练习教程 + 买菜清单 + 成果记录，配了一个手机网页（PWA），加到主屏幕就像 APP 一样用。

## 📱 手机页面能干什么

- **🛒 清单**：自动从教程里提取每周的采购清单，超市里买一样勾一样（勾选存在手机里，**没信号也能看**）
- **📖 教程**：手机排版的完整练习教程
- **📷 成果**：做完拍照发到本仓库的 Issues，这页自动变成成果相册
- **⭐ 收藏**：我喜欢的菜品和原料清单

## 🚀 一次性开通（只需做一次）

1. **把仓库设为公开**：仓库页面 → Settings → 拉到最底下 Danger Zone → Change visibility → Make public
   （免费版 GitHub Pages 要求仓库公开。公开后菜谱和做饭照片别人也能看到，介意的话不要放隐私内容。）
2. **把改动合并到 main 分支**（合并后 Actions 会自动部署，第一次约 1 分钟）
3. 打开网址：**https://cxy0714.github.io/cook_learning/** ✅

### 加到手机主屏幕（像装了个 APP）

- **iPhone（Safari 打开网址）**：点底部「分享」按钮 → 「添加到主屏幕」
- **安卓（Chrome/Edge 打开网址）**：右上角菜单 → 「添加到主屏幕」或「安装应用」

## 📷 怎么记录成果

1. 手机装免费的 **GitHub App** 并登录（只需一次）
2. 做完饭，在页面「成果」页点 **「记录今天的成果」**（或在 App 里：本仓库 → Issues → New issue）
3. 标题写日期和菜名，正文点图片按钮**直接拍照贴进去**，提交
4. 回到页面「成果」页，自动显示 ✅

## ✏️ 怎么改清单

直接编辑 `Food/practice_guide.md`。页面会自动提取每个 `## 第 N 周末` 小节里以
`**采购**：` 开头的行，把「、」分隔的每一项变成可勾选项，整段「（……）」会显示为提示而不是勾选项。

例如：

```markdown
**采购**：番茄 3 个、鸡蛋 1 盒、（油和盐家里有就不买）
```
