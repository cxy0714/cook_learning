# 下饭炒菜速成（3 个周末搞定你馋的三道）

> 你选了炒菜——选得好。**炒菜是地基**：学会这一套手法，以后面食、炖菜都是变体。
> 目标菜：番茄炒蛋 → 青椒炒肉 → 酸辣土豆丝（由易到难，正好排成 3 周）。

---

## ⭐ 先记这个：炒菜万能公式

**几乎所有炒菜都是这 6 步，换个料就是另一道菜。**

1. **备料**：所有菜先切好、调料先摆到手边。（新手手忙脚乱 90% 是因为边炒边找东西。一定先备齐再开火。）
2. **热锅凉油**：空锅烧到微微冒烟 → 再倒油。这样不粘锅。
3. **爆香**：油热了下蒜末（葱、姜），闻到香味，几秒就行。
4. **下主料**：**耐熟的先下**（肉、土豆），**易熟的后下**（鸡蛋、叶菜）。
5. **调味**：盐 / 生抽 / 一点糖，快速翻炒均匀。
6. **判断熟 → 出锅**：照下面的熟度口诀，对了就关火盛出。

> **新手两大坑 & 解法**
> - **粘锅** → "热锅凉油"（锅先烧热再倒油）就解决了。
> - **炒老 / 没熟** → 蛋和肉**先单独炒到刚好、盛出来**，最后再倒回锅里拌一下。分开炒，谁都不会过火。这是这三道菜共用的关键技巧。

**熟度速查**：蛋——不流动、金黄就好（宁可早出锅）；肉——切开全白不见粉红；叶菜——变软塌（很快，1–2 分钟）；土豆丝——断生还带脆就出锅。

---

## 第 1 周末：番茄炒蛋 + 清炒青菜 + 青椒炒肉（加菜）

学**炒蛋**和**"先炒盛出再回锅"**这个核心套路。

### 番茄炒蛋（本周主练，做两遍）
**备料**：鸡蛋 3 个打散、加一小撮盐打匀；番茄 2 个切块；蒜切末。
1. 热锅凉油，油热，**倒蛋液**——别急着翻，等边缘鼓起凝固，再推炒几下，**整体凝固金黄就立刻盛出**（这时蛋是嫩的，余温还会熟，别炒过）。
2. 锅里补一点油，下番茄块，炒到**出红汁、变软**（2–3 分钟）。汁少可加两勺水。
3. **倒回炒蛋**，加盐 +**一点点糖**（番茄菜加糖更好吃），翻匀 30 秒，出锅。

### 清炒青菜（30 秒上手，当兜底）
热锅凉油 → 蒜末爆香 → 青菜大火快炒 → **变软塌**加盐 → 出锅。全程 1–2 分钟。

### 加菜：青椒炒肉（馋了就做，提前解锁）

你馋大青椒——那就提前做！建议放**周日**（周六先把番茄炒蛋练熟，手法是同一套）。一次不完美没关系，第 2 周还会正式再练它一遍。

**备料**：肉（鸡腿肉 / 猪里脊）切片，加**生抽 + 料酒 + 一勺淀粉**抓匀腌 10 分钟（这步让肉嫩、不柴）；大青椒切块；蒜末。
1. 热锅凉油，下肉片炒到**全部变白**（切开看：不见粉红才算熟），盛出。
2. 锅里补点油，下蒜末、青椒块，炒到青椒变软。
3. **倒回肉**，加盐 + 生抽，翻匀 30 秒，出锅。

**采购**：番茄 3 个、鸡蛋 1 盒、青菜 1 把、大青椒 4–5 个、鸡腿肉或猪里脊 1 块、料酒、淀粉、（油、盐、糖、生抽、蒜——哥家有就不买）

---

## 第 2 周末：青椒炒蛋 → 青椒炒肉

复用炒蛋，再升级到**炒肉**（你馋的）。关键学**肉上浆**和**肉先炒盛出**。
（第 1 周加菜时已经做过一次的话，这周就是复盘：翻出你上次记的问题，对着改，把它做顺。）

### 青椒炒蛋（热身，练切青椒）
青椒切块。炒蛋盛出 → 炒青椒到变软 → 倒回蛋 → 加盐生抽 → 出锅。

### 青椒炒肉（重点）
**备料**：肉（鸡腿肉 / 猪里脊）切片，加**生抽 + 料酒 + 一勺淀粉**抓匀腌 10 分钟（这步让肉嫩、不柴）；青椒切块；蒜末。
1. 热锅凉油，下肉片炒到**全部变白**，盛出。
2. 锅里下蒜末、青椒，炒到青椒变软。
3. **倒回肉**，加盐 + 生抽，翻匀，出锅。

**采购**：大青椒 4–5 个、鸡腿肉或猪里脊 1 块、鸡蛋（续）、（料酒、淀粉第 1 周买过就还有，不用重买）

---

## 第 3 周末：酸辣土豆丝（毕业菜）

最考火候的一道，放最后。学**切丝 / 刨丝、泡水、大火保脆**。

**备料**：土豆切细丝（用**刨丝器**最省事）→ **泡清水冲掉淀粉**（这步是"脆"的关键，不泡会黏成一坨）→ 沥干；干辣椒切段；蒜末。
1. 热锅凉油，下干辣椒、蒜末爆香。
2. 下土豆丝，**全程大火快炒**。
3. 沿锅边淋**醋**（陈醋 / 香醋），加盐。
4. 炒到**断生、还带点脆**就出锅——**别炒到软塌**，软了就不是酸辣土豆丝了。全程也就 2–3 分钟。

**采购**：土豆 3 个、干辣椒、陈醋、（刨丝器，哥家没有就买一个，几块钱）

---

## 本周加练：虾仁炒西兰花 + 青椒炒牛肉

馋什么做什么——这两道顺便把**"焯水"**和**"牛肉嫩"**两个新技能练上。
**三句口诀贴墙上：蒜姜先下、肉/虾炒变色就盛出、全程不加水。**

### 虾仁炒西兰花（虾仁 150g / 西兰花 300g）
**新技能：西兰花先焯水**——光靠炒炒不透、还发硬。
**备料**：西兰花切小朵，烧一锅水加点盐和油，焯 1–2 分钟变翠绿就捞出、**过一下凉水**（保脆保绿）、沥干；虾仁加**少许盐、白胡椒、料酒、一小勺玉米淀粉**抓匀腌 10 分钟；蒜切末。
1. 热锅凉油，下虾仁炒到**刚变粉红、打卷就盛出**（1–2 分钟，炒老就硬）。
2. 补点油，**先下蒜末爆香**。
3. 下西兰花**大火快炒**半分钟到一分钟（已焯熟，主要炒上味）。
4. **倒回虾仁，加盐**，翻匀出锅。**不加水。**
> 这道清淡鲜甜，别放十三香辣椒面；想提鲜加一点点蚝油就够。

### 青椒炒牛肉（青椒 300g / 牛肉条 200g）
**备料**：牛肉条加**生抽 + 料酒 + 玉米淀粉 + 少许盐**（香油一点点就够）腌 15 分钟；青椒切条；蒜姜切末；调料摆手边。
1. 热锅凉油，**油别太多**，温热微冒烟下牛肉条，**快炒到变色就盛出**（约 1 分钟，别炒几分钟会老）。
2. 补点油，**先下蒜姜末爆香**（爱辣丢两段干辣椒）。
3. 下青椒**大火**炒到变软、起虎皮。
4. **倒回牛肉，加盐 + 生抽**（想要点甜加少许糖），快速翻匀出锅。**全程不加水。**

**采购**：虾仁 150g、西兰花 1 颗、青椒 300g、牛肉条 200g、（生抽、料酒、淀粉、蒜、姜、白胡椒、蚝油——有就不用买）

## 三周后

这三道吃透，你就掌握了炒菜的完整套路。之后想学**面食 / 咖喱 / 牛排**，跟我说一声，我按同样的"公式 + 分步 + 采购"给你接着排——手法都是通的，不用重学。

**每次练完记一笔**哪步出了问题（粘锅了？蛋老了？土豆软了？），下次改。进步会很快。

---

## 📷 拍照小贴士

一句话：**干净背景 + 填满画面 + 一撮绿色。**

1. **擦干台面 / 案板**再摆——水渍反光最毁图。
2. **拿走背景杂物**（蛋托、酱油瓶、纸巾）。
3. **靠窗自然光、侧光**拍，别开顶灯直射或闪光。
4. **主料（肉 / 虾）翻几块到面上**、堆出高度，别全埋在底下。
5. **撒点葱花 / 香菜 / 白芝麻**提亮，**趁热拍**。

**裁剪**：碗装、面条用**俯拍 + 正方形（1:1）**；有高度、有层次的菜用 **45° + 竖构图（4:5）**。裁掉边角杂物，让碗占画面约 80%。

## 📝 练习记录

### 6/13 周六：炒蛋练手 + 蒜苔炒牛肉 + 青椒炒鸡蛋 + 红薯叶挂面
和哥哥一起，两人份，第一次做这么多——赞！**做得好**：牛肉腌到位（酱油 + 香油 + 料酒 + 玉米淀粉 + 盐，腌 15–20 分钟）。

**下次重点改 3 件事：**
1. **炒菜不加水** —— 两道菜最后都加了水煮，炒变煮。炒菜要大火、快、不放水。（想做面浇头是另一种做法：勾芡收汁，不是加水煮。）
2. **蒜姜先下锅爆香** —— 两道都加晚了 / 忘了。油热先下蒜姜，再下主料。
3. **少放复合调料** —— 十三香 / 辣椒面 / 胡椒粉先收起来，盐 + 生抽为主，先吃食材本味。

**小提醒**：油别太多（被溅到手，注意安全）；青椒炒蛋记得"先炒蛋盛出、最后回锅"；煮面转中火 + 锅盖留条缝就不容易扑锅（你点凉水的做法是对的）。
