## 蒜苔炒牛肉
**做过**：6/13 周六 · 两人份
**用量**：蒜苔 200g、牛肉条 200g
**腌料**：酱油 + 芝麻香油 + 料酒 + 玉米淀粉 + 少许盐，抓匀腌 15–20 分钟
**调料**：盐、生抽（这次还加了十三香 / 辣椒面 / 胡椒粉，下次省掉）
**做法**：热锅凉油 →（蒜姜爆香）→ 牛肉条炒到变色盛出 → 炒蒜苔 → 倒回牛肉 → 加盐 + 生抽翻匀出锅

**备注**
- ✅ 牛肉腌得到位，嫩
- ⚠️ 油放多了溅到手 → 下次少油，食材贴着锅边滑下去
- ⚠️ 忘了蒜姜、最后加水变"煮菜" → 蒜姜先下锅、全程不加水
- ⚠️ 调料太杂盖住本味 → 只留盐 + 生抽

## 青椒炒鸡蛋
**做过**：6/13 周六 · 两人份
**用量**：鸡蛋 5 个、青椒约 300g
**配料**：鸡蛋打散 + 少许盐拌匀
**调料**：盐、生抽（这次也加了十三香 / 辣椒面 / 胡椒粉，下次省掉）
**做法（正确版）**：热锅凉油 → 先炒蛋到金黄盛出 →（蒜姜爆香）→ 炒青椒到变软 → 倒回蛋 → 加盐 + 生抽翻匀出锅

**备注**
- ⚠️ 这次先炒的青椒、顺序反了 → 记住"先炒蛋盛出、最后回锅"
- ⚠️ 蒜姜加晚了 → 油热先下蒜姜
- ⚠️ 最后加水煮了 → 炒菜不加水

## 炒鸡蛋（纯练手）
**做过**：6/13 周六
**用量**：鸡蛋 2 个
**做法**：打散 + 一小撮盐 → 热锅凉油 → 倒蛋液，边缘凝固后推炒 → 整体金黄就盛出

**备注**
- 练"嫩"的手感——宁可早出锅，余温还会再熟一点，别炒过
